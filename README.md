# TodoList REST API

Простой RESTful сервис для управления списком задач (TodoList), реализованный на **Spring Boot**. Проект подходит для изучения основ создания REST API с использованием Spring, JPA, H2 и Maven.

## Функциональность

- Создание задачи
- Получение списка всех задач
- Получение задачи по ID
- Обновление задачи
- Удаление задачи

## Технологии

- **Java 17** – язык программирования
- **Spring Boot 3.2.x** – фреймворк
- **Spring Data JPA** – работа с базой данных
- **H2 Database** – встроенная in-memory БД
- **Maven** – система сборки
- **Lombok** (опционально) – сокращение кода

### API Endpoints


| Метод  | Эндпоинт          | Описание              |
|--------|-------------------|-----------------------|
| GET    | `/api/todos`      | Получить все задачи   |
| GET    | `/api/todos/{id}` | Получить задачу по ID |
| POST   | `/api/todos`      | Создать задачу        |
| PUT    | `/api/todos/{id}` | Обновить задачу       |
| DELETE | `/api/todos/{id}` | Удалить задачу        |
